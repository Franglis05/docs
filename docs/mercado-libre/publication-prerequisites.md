# Pre-Requisitos de Publicación de Mercado Libre

## Descripción

La verificación de **Pre-Requisitos de Publicación** permite validar que un producto cumple con todos los requisitos necesarios antes de ser publicado en Mercado Libre. Esto evita errores y rechazos durante el proceso de publicación.

## ¿Cuándo se utiliza?

Se utiliza antes de intentar publicar un producto en Mercado Libre para asegurar que cumple con todos los requisitos.

Casos típicos:
- Validación previa a la primera publicación de un producto
- Verificación después de modificar datos del producto
- Diagnóstico de errores de publicación

## Acceso

Gestión de Materiales → Productos → Producto → Pestaña Mercado Libre

## Campos principales

### Requisitos Obligatorios

- **Condición del Producto**
  Estado del producto: Nuevo o Usado.

- **Tipo de Publicación**
  Nivel de visibilidad: Bronce, Plata o Gratuita.

- **Precio del Producto**
  Precio de venta establecido en la lista de precios.

- **Existencia del Producto**
  Stock disponible (debe ser mayor a 0).

- **Imágenes**
  Al menos una imagen en formato PNG, JPG o WEBP.

- **Categoría de Mercado Libre**
  Categoría asignada del marketplace.

- **Atributos Obligatorios**
  Atributos requeridos por la categoría seleccionada.

## Flujo del proceso

### 1. Acceder a la ventana de producto

Abrir la ventana de producto y seleccionar el registro del producto a publicar.

![Ventana de producto](/assets/img/docs/mercado-libre/d13d516404c240219df4b26735659688.png)

### 2. Configurar los requisitos del producto

Verificar y completar los requisitos obligatorios del producto.

![Requisitos del producto](/assets/img/docs/mercado-libre/16bda63d21fe4979a8642b0d20685426.png)

| Requisito | Descripción |
|-----------|-------------|
| Condición del producto | Nuevo o usado |
| Tipo de publicación | Bronce, plata o gratuita |
| Precio del producto | Debe estar establecido |
| Existencia del producto | Debe ser mayor a 0 |

### 3. Verificar imágenes del producto

Confirmar que el producto tenga imágenes en formatos aceptados: PNG, JPG o WEBP.

![Verificación de imágenes](/assets/img/docs/mercado-libre/996faaeb3bb44e588c2192dea36cb3b8.png)

### 4. Configurar atributos y categoría

Completar los atributos del producto y verificar la categoría asignada de Mercado Libre.

![Atributos y categoría](/assets/img/docs/mercado-libre/b41de473b04f4cd68379c0e666228689.png)

### 5. Asignar el producto

Verificar que el producto esté correctamente asignado en la categoría correspondiente.

![Asignación de producto](/assets/img/docs/mercado-libre/89894cdab30a43dd87b3df9e4ee767f6.png)

### 6. Intentar publicar el producto

Proceder a publicar el producto y verificar si se presenta algún mensaje de error.

![Publicación del producto](/assets/img/docs/mercado-libre/796b9a4dedf74bf4b68355cca7d130ba.png)

## Consideraciones importantes

- Verificar todos los requisitos antes de intentar publicar
- El stock debe ser mayor a 0 para poder publicar
- Las imágenes deben estar en formatos válidos (PNG, JPG, WEBP)
- Los atributos obligatorios varían según la categoría seleccionada

## Lista de Verificación

Antes de publicar, confirmar que el producto cumpla con:

- [ ] Precio establecido
- [ ] Stock mayor a 0
- [ ] Condición definida (nuevo/usado)
- [ ] Tipo de publicación seleccionado
- [ ] Al menos una imagen en formato válido
- [ ] Categoría de Mercado Libre asignada
- [ ] Atributos obligatorios completados
