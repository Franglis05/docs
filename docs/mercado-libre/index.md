# Integración con Mercado Libre

## Descripción

La integración con **Mercado Libre** permite gestionar la publicación y actualización de productos desde Solop ERP hacia el marketplace. Incluye la configuración inicial, gestión de productos y manejo de errores.

## ¿Cuándo se utiliza?

Se utiliza cuando se requiere vender productos a través de Mercado Libre desde Solop ERP.

Casos típicos:
- Configuración inicial de la integración
- Publicación de nuevos productos
- Actualización de precios y stock
- Gestión de estados de publicación
