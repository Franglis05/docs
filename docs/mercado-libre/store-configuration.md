# Configuración de Tienda en Mercado Libre y Gestión de Stock

## Descripción

La ventana de **Configuración de Tienda** permite establecer los parámetros necesarios para conectar Solop ERP con una cuenta de Mercado Libre. Incluye la configuración del almacén, lista de precios y URLs de conexión para categorías y atributos.

## ¿Cuándo se utiliza?

Se utiliza cuando se necesita configurar una nueva tienda de Mercado Libre o actualizar la configuración existente.

Casos típicos:
- Alta de una nueva tienda de Mercado Libre
- Modificación del almacén asociado para gestión de stock
- Actualización de listas de precios
- Configuración de URLs para categorías y atributos

## Acceso

Integración Mercado Libre → Configuración → Tienda Web

## Campos principales

### Pestaña General

- **Nombre de la Tienda**
  Identificador de la tienda dentro del sistema.

- **Dirección Web**
  URL de la tienda en Mercado Libre.

- **Tipo de Tienda**
  Clasificación de la tienda (comercial, vendedor, etc.).

- **Almacén**
  Almacén asociado para reflejar el stock disponible en las publicaciones.

- **Lista de Precios**
  Lista de precios a utilizar para la publicación de productos.

- **URL Categoría**
  URL de N8N para la asignación de categorías.

- **URL Atributos**
  URL de N8N para la asignación de atributos.

## Flujo del proceso

### 1. Abrir la Ventana de Tienda

Acceder a la ventana de Tienda Web y seleccionar el tipo de tienda Mercado Libre.

![Ventana de tienda](/assets/img/docs/mercado-libre/c8f266011290415fbc563b8624a5eceb.png)

### 2. Configurar Detalles de la Tienda

Introducir el nombre de la tienda y proporcionar la dirección web.

![Detalles de tienda](/assets/img/docs/mercado-libre/f147713efe614f4bab4e62aca2d986e9.png)

### 3. Establecer Tipo de Tienda y Almacén

Seleccionar el tipo de tienda y asegurar que el almacén esté correctamente configurado para reflejar el stock disponible.

![Tipo y almacén](/assets/img/docs/mercado-libre/cb91b2cfbbae43919f224c0b9b679c23.png)

### 4. Configurar Lista de Precios

Introducir la lista de precios que se utilizará para la publicación de productos.

![Lista de precios](/assets/img/docs/mercado-libre/0706362905204c8c9b9d3841e08bab49.png)

### 5. Configurar URL para Categoría y Atributos

Introducir las URLs de N8N para la correcta asignación de categorías y atributos.

![URLs de configuración](/assets/img/docs/mercado-libre/8434f75d8d8341e1a69d7a92ea594a85.png)

## Consideraciones importantes

- Verificar que toda la información ingresada sea correcta antes de guardar
- Asegurar que el stock en el almacén coincida con lo que se publicará en la tienda
- Utilizar plantillas para la lista de precios para facilitar la carga de datos
- Revisar periódicamente el stock y los precios para mantener la tienda actualizada
