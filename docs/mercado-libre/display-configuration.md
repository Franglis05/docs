# Configuración de Visualización para Mercado Libre

## Descripción

La ventana de **Definición de Visualización** permite configurar cómo se mostrarán los productos en Mercado Libre. Establece la relación entre los productos del ERP y su presentación en el marketplace.

## ¿Cuándo se utiliza?

Se utiliza cuando se necesita configurar la visualización de productos para Mercado Libre.

Casos típicos:
- Configuración inicial de visualización para nuevos productos
- Verificación de registros existentes de Mercado Libre
- Asignación de categorías de tributo

## Acceso

Integración Mercado Libre → Configuración → Definición de Visualización

## Flujo del proceso

### 1. Abrir la ventana de definición de visualización

Acceder a la plataforma y buscar la ventana de definición de visualización. Verificar que exista un registro específico para Mercado Libre.

![Ventana de visualización](/assets/img/docs/mercado-libre/68a0eb5f2d24427caa06001ab9c181ab.png)

### 2. Verificar registros existentes

Revisar los registros disponibles en la ventana de definición y confirmar que tengan las características necesarias para la configuración.

![Registros existentes](/assets/img/docs/mercado-libre/68a0eb5f2d24427caa06001ab9c181ab.png)

### 3. Configurar el producto

Seleccionar el tipo de visualización tienda que apunta al producto, asegurar que esté correctamente nombrado como 'Mercado Libre' y proceder a la configuración de las categorías de tributo.

![Configuración de producto](/assets/img/docs/mercado-libre/bfaf3a28dae84a6b9ef7ae584e295171.png)

## Consideraciones importantes

- Verificar que todos los registros sean correctos antes de proceder con la configuración
- Confirmar que el nombre del producto sea exacto para evitar confusiones
- Mantener un registro de los productos ya configurados para evitar duplicaciones
- Utilizar plantillas para la configuración de productos similares
