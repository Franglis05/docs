# Publicación de Producto en Mercado Libre

## Descripción

El proceso de **Publicación de Producto** permite enviar un producto desde Solop ERP hacia Mercado Libre, creando una publicación activa en el marketplace. Incluye la validación de requisitos, selección del tipo de publicación y verificación del resultado.

## ¿Cuándo se utiliza?

Se utiliza cuando un producto está listo para ser publicado en Mercado Libre.

Casos típicos:
- Primera publicación de un nuevo producto
- Republicación de un producto previamente pausado
- Actualización de una publicación existente

## Acceso

Integración Mercado Libre → Publicaciones → Publicar Producto

## Acciones disponibles

- **Publicar Producto**
  Envía el producto a Mercado Libre creando una nueva publicación.

- **Procesar Publicación**
  Ejecuta el proceso de sincronización con Mercado Libre.

- **Verificar Publicación**
  Abre el enlace de la publicación para confirmar su estado.

## Flujo del proceso

### 1. Validación de Requisitos del Producto

Verificar que el producto cumpla con los requisitos: precio, existencia, tributos, categoría y condición.

![Validación de requisitos](/assets/img/docs/mercado-libre/93c1dfd7899b47aeac4168300ef22888.png)

### 2. Selección del Tipo de Publicación

Elegir el tipo de publicación según la visibilidad deseada.

![Selección de tipo](/assets/img/docs/mercado-libre/ac3815cf54844a478ed4ba1b95ca3887.png)

| Tipo | Descripción |
|------|-------------|
| Plata | Mayor visibilidad, con costo |
| Bronce | Visibilidad media, costo reducido |
| Gratuito | Sin costo, visibilidad básica |

### 3. Proceso de Publicación del Producto

Ejecutar el proceso de publicar producto: seleccionar la tienda web (Mercado Libre) y confirmar la acción.

![Proceso de publicación](/assets/img/docs/mercado-libre/35d71ca96b7a451487adacf8a3497454.png)

### 4. Verificación de Publicaciones

Revisar las publicaciones en la ventana correspondiente, filtrando según el estatus: Pausadas, Activas, En proceso o Por verificar.

![Verificación de publicaciones](/assets/img/docs/mercado-libre/b94050f5de6a484b9609557b0487d1f8.png)

### 5. Guardar Registro de Publicación

Confirmar que se guarde el registro de publicación con el precio y atributos del producto.

![Registro de publicación](/assets/img/docs/mercado-libre/b5cbb8d40bed4a6d9cefc8fb8d0b45bb.png)

### 6. Procesar la Publicación

Hacer clic en 'Procesar publicación' y refrescar la página para verificar el estado.

![Procesamiento](/assets/img/docs/mercado-libre/de14a21887644a0ab5333447b653564c.png)

### 7. Verificación Final de la Publicación

Abrir el enlace proporcionado para verificar que el producto esté visible en Mercado Libre con los atributos y precio correctos.

![Verificación final](/assets/img/docs/mercado-libre/de14a21887644a0ab5333447b653564c.png)

## Consideraciones importantes

- Completar todos los requisitos antes de proceder con la publicación
- Verificar que la URL de publicación sea correcta antes de compartirla
- Mantener un checklist de los requisitos para facilitar la validación
- El proceso puede tardar unos minutos en reflejarse en Mercado Libre

## Ejemplo de uso

Para publicar un producto:
1. Verificar que cumpla con todos los pre-requisitos
2. Seleccionar tipo de publicación (ej. Plata)
3. Ejecutar proceso "Publicar Producto"
4. Seleccionar tienda "Mercado Libre"
5. Confirmar y esperar procesamiento
6. Verificar publicación en Mercado Libre
