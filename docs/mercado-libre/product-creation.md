# Creación y Configuración de un Producto en Mercado Libre

## Descripción

El proceso de **Creación de Producto** permite registrar nuevos productos en Solop ERP con todos los atributos necesarios para su publicación en Mercado Libre. Incluye la asignación automática de categorías sugeridas por el marketplace.

## ¿Cuándo se utiliza?

Se utiliza cuando se necesita crear un nuevo producto que será publicado en Mercado Libre.

Casos típicos:
- Alta de nuevos productos para venta en Mercado Libre
- Configuración de atributos específicos del marketplace
- Asignación de categorías sugeridas por Mercado Libre

## Acceso

Gestión de Materiales → Productos → Producto

## Campos principales

### Pestaña Producto

- **Nombre del Producto**
  Nombre descriptivo que se mostrará en la publicación.

- **Unidad de Medida**
  Unidad en la que se vende el producto.

- **Categoría**
  Categoría interna del producto en el ERP.

### Pestaña Mercado Libre

- **Categoría Sugerida**
  Categoría de Mercado Libre asignada automáticamente.

- **Modelo**
  Modelo del producto (atributo obligatorio).

- **Marca**
  Marca del producto (atributo obligatorio).

- **Color**
  Color del producto (si aplica).

## Flujo del proceso

### 1. Abrir la ventana de producto

Acceder a la ventana de producto, seleccionar la opción para crear un nuevo producto, introducir el nombre y establecer las unidades de medida y categoría.

![Ventana de producto](/assets/img/docs/mercado-libre/05a200912cca485b9470899e60515bdf.png)

### 2. Asignar una categoría

En la ventana de producto, buscar la opción para sugerir categorías y seleccionar 'Mercado Libre'. Revisar las categorías sugeridas y seleccionar la más adecuada.

![Asignación de categoría](/assets/img/docs/mercado-libre/182fbef4b18e4a9f8b759f85f0d9c0e2.png)

### 3. Revisar y actualizar atributos del producto

Verificar que se hayan creado los atributos necesarios para la publicación. Completar los atributos obligatorios como modelo, marca y color.

![Atributos del producto](/assets/img/docs/mercado-libre/d2f9fdbc1f184d7ebb5433675662a8fc.png)

### 4. Finalizar la configuración del producto

Una vez que todos los atributos estén completos, hacer clic en 'ok' para actualizar los atributos del producto y confirmar la configuración.

![Finalización de configuración](/assets/img/docs/mercado-libre/734e102ebb254b41b21c8f7371abca11.png)

## Consideraciones importantes

- Completar todos los atributos obligatorios antes de finalizar la configuración
- Revisar las categorías sugeridas para seleccionar la más relevante
- Utilizar descripciones claras y concisas para facilitar la sugerencia de categorías
- Mantener un registro de los atributos más comunes para productos similares

## Ejemplo de uso

Para crear un producto "Balón de Fútbol":
1. Acceder a la ventana de producto y crear nuevo registro
2. Ingresar nombre "Balón de Fútbol", unidad "Unidad", categoría "Deporte"
3. Solicitar categoría sugerida de Mercado Libre
4. Completar atributos: Marca, Modelo, Color
5. Guardar el producto
